# Forex Education Bot

A simple Telegram bot for Forex trading education and calculations.

## Features
- Position Size Calculator
- Pip Value Calculator  
- Risk-Reward Ratio Calculator

## Commands
- `/start` - Start the bot
- `/help` - Show help
- `/position_size [balance] [risk%] [stop_loss]` 
- `/pip_value [pair] [lot_size]`
- `/risk_reward [entry] [stop] [take]`

## Deployment
1. Get token from @BotFather
2. Add TELEGRAM_TOKEN to Railway variables
3. Deploy!
