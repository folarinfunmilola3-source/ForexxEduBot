# Forex Education Bot 📊

An educational Telegram bot designed to help users learn Forex trading through interactive tools and calculators.

## Features

✅ **Calculators**
- Position Size Calculator
- Pip Value Calculator
- Risk-Reward Ratio Calculator
- Pivot Points Calculator

✅ **Market Analysis**
- Economic Calendar
- Market Sentiment Overview

✅ **Education**
- Forex Basics
- Candlestick Patterns
- Technical Indicators
- Risk Management
- Interactive Quiz

## Deployment

1. Get a bot token from @BotFather on Telegram
2. Set `TELEGRAM_TOKEN` as an environment variable
3. Deploy on Railway with this repository

## Commands

- `/start` - Welcome message
- `/help` - Show all commands
- `/education` - Learn Forex topics
- `/position_size` - Calculate position size
- `/pip_value` - Calculate pip value
- `/risk_reward` - Calculate R:R ratio
- `/pivot_points` - Calculate pivot levels
- `/economic_calendar` - View economic events
- `/sentiment` - Market sentiment
- `/quiz` - Take a Forex quiz

## Disclaimer

⚠️ This bot is for educational purposes only. It does not provide financial advice or real trading signals. All calculations are simulations.
