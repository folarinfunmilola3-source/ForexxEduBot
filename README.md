# Forex Education Bot

A simple educational Telegram bot for Forex trading calculations.

## Commands
- `/start` - Start the bot
- `/help` - Show help
- `/position_size` - Calculate position size
- `/pip_value` - Calculate pip value
- `/risk_reward` - Calculate risk-reward ratio

## Deployment
1. Get token from @BotFather
2. Set TELEGRAM_TOKEN environment variable
3. Deploy on Railway
